コンソール上で Object の中身を確認したい場合は `console.dir(){:js}` を使う。第2引数としてオプションを渡せるので、全レベルを表示したい場合は `{ depth: null }{:js}` を渡す。

```js
console.dir(object, { depth: null });
```

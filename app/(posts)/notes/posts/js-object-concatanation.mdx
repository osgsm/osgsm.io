JavaScriptでオブジェクトを連結したい場合は スプレッド構文（spread syntax）  `...{:js}` を使う。

同じプロパティ名がある場合は後者の値で上書きされるので要注意。


```js
const firstPerson = {
    name: "Sam",
    age: 18
}

const secondPerson = {
    age: 25,
    type: "admin"
}

const mergedObjects = {...firstPerson, ...secondPerson};
console.log(mergedObjects); // {name: "Sam", age: 25, type: "admin"}
```

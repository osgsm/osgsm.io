JavaScript の `%` 剰余演算子（remainder operator）を使うことで、その結果から、ある数値が偶数か奇数かを判定できる。

`n % 2` とすることで、結果が `0` であれば偶数、 `1` であれば奇数になる。

```js
const evenOrOdd = (number) => number % 2 === 0 ? 'even' : 'odd';
// evenOrOdd(2) -> 'even'
// evenOrOdd(3) -> 'odd'
```

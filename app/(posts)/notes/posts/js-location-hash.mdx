JavaScript で URL のハッシュ部分を取得するには `Location` インターフェースの `hash` プロパティを使う。値は文字列になる。

```js title="https://osgsm.io/notes#2 の場合"
window.location.hash
// "#2"
```

`#` よりあとの文字列を取得したい場合は `substring(1){:js}` を使う。数値として扱う場合はそれを `Number(){:js}` に渡す。

```js
Number(window.location.hash.substring(1));
// 2
```

JavaScript  でローカルストレージに特定のキーを持つアイテムがあるかどうかを調べるには、 `in` operator を使う。

```js
console.log('theme' in localStorage);
// true or false
```

ここでは `theme` というアイテムがローカルストレージに保存されていれば `true` を返し、そうでなければ `false` を返す。

`DOMTokenList.contains(){:js}` メソッドを使うことで、ある要素が特定の class を持っているかどうかをチェックできる。返り値は boolean。

```html
<div id="box" class="activated"></div>
```

```javascript
const box = document.querySelector("#box");
box.classList.contains("activated"); // true
box.classList.contains("highlighted"); // false
```
